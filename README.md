## 기능 구현 목록

> 핵심 기능 먼저 구현 후 세부 기능 확장하기

- [ ] 제목을 입력할 수 있다. (글자수 제한: 10자)

- [ ] 내용을 입력할 수 있다. (글자수 제한: 15자)

- [ ] 랜덤 배경 색상 버튼을 클릭하면 랜덤으로 색상을 지정할 수 있다.
